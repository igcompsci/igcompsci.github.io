Getting Started

这里是我个人整理的一些关于 CIE 考试局 IGCSE Computer Science 学科的资料和工具。目前内容不多，主要是一些我自己编写的在线工具，方便学生在线练习，强化知识理解，后续会陆续更新更多有用的内容。

做这个网站的初衷是因为国内关于这一块的内容太少了，希望我写的这些内容能对你们多少有些帮助。

重点：为了方便同学们练习伪代码 (正常伪代码是不能运行的)，我写了 IGCSE PSEUDOCODE COMPILER 这个小工具，可以在网页上直接运行伪代码，这样就能看到运行结果，验证代码的正确性，非常地方便。

Roadmap

  • 完善 igpc, 支持更多特性 (多变量声明, 类型检查, 文件...)
  • 实现 igpc editor, 方便在线编辑和运行伪代码
  • 网页版 CPU 模拟器 (Larry's Little Computer)
  • 网页版 SQL 练习工具 (导入往年考题的数据表)
  • 待定：计算 "布尔表达式" 的真值表
  • 发布 Paper 2 的复习笔记
